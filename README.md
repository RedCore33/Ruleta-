## Ejecución
El proyecto se compila, testea y activa gracias al [MakeFile](makefile) creado. Opciones disponibles:
- 'clean' -> Limpia el proyecto de archivos generados por .NET
- 'run' -> Ejecuta el programa
- 'test' -> Ejecuta los test(s)
